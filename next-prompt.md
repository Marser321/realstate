Create a visually immersive Property Detail page.

**DESIGN SYSTEM (REQUIRED):**
[Use the existing "Glass & Gloss" system concepts from DESIGN.md]
- **Media First**: The layout should be dominated by high-resolution imagery.

**Page Structure:**
1. **Header**: Transparent/Glass overlay on top of the hero image.
2. **Hero**: Full-width image gallery masonry or hero slider.
3. **Main Content (Two Columns)**:
    - **Left (Details)**: Description, Features Grid (Icons), Location Map, "Lifestyle Match" Score.
    - **Right (Sticky Sidebar)**: Agent Card (floating), "Schedule Viewing" Form, Price.
4. **Similar Properties**: Carousel at the bottom.
